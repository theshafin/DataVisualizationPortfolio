Dataset can be found under:  
(Images/dataset.csv) 


## Overview  
This dataset provides records of breeding bird pairs observed across multiple regions of Alberta between 2000 and 2020. It includes counts of nesting pairs for various bird species, with a focus on population changes, species diversity, and habitat distribution over time. The goal of these visualizations is to explore how breeding patterns have evolved across different locations and years, highlight regional differences in species abundance, and assess long-term ecological trends in Albertaâ€™s avian populations.  


## Tools  
- Tableau & Microsoft Excel  
- Built-in Tableau features for data cleaning, filtering, and aggregation  
- Microsoft Excel for preliminary data preparation and normalization  
- Dataset provided by the City of Calgary
